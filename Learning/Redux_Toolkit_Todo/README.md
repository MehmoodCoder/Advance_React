# 📝 React Tailwind v4 - Redux Toolkit Todo Application

A modern, lightweight, and highly interactive Todo Application built using React.js and Tailwind CSS v4. This project manages application layouts globally using the Redux Toolkit engine, completely bypassing old prop-drilling models.

---

## ✨ Features

- 🌐 **Advanced Global Storage** – Integrates Redux Toolkit architecture to synchronize record pipelines instantly across decoupled form hooks and list segments.
- 🎨 **Tailwind v4 Dynamic Shell** – Implements a premium responsive dark theme wrapped inside a structured micro-grid background matrix.
- 📊 **Real-time Vector Tracking** – Monitors stack lengths interactively to keep live counts of active repository configurations visible.
- ⚡ **Optimized Data Reducers** – Utilizes high-performance pure functions to manage localized updates without mutating the core application state.

---

## 🛠️ Tech Stack

* **Framework:** React.js (Vite Template), Redux, RTK
* **Styling:** Tailwind CSS
* **Language:** JavaScript (ES6+), HTML

---

## 🎯 Component & Slice Architecture Overview

### Core Architecture Components

| Component / File | Purpose |
|------------------|---------|
| **store.js** | Configures the central warehouse ecosystem to process global data distribution uniformly. |
| **todoSlice.js** | Manages application state models alongside dedicated modular reducer engine functions. |
| **AddTodo.jsx** | Captures input field triggers and transmits dynamic payloads straight to storage branches. |
| **Todos.jsx** | Maps ongoing record collections to trigger targeted state deletions and content updates. |
| **App.jsx** | Serves as the global shell interface, embedding clean grid matrices across screen viewports. |

## 🚀 Core Concepts Covered

- **`configureStore()`**: Establishing a centralized global warehouse to aggregate multiple state slices into a single source of truth.
- **`createSlice()`**: Bundling initial data models alongside localized object reducer worker functions uniformly.
- **`useDispatch()`**: Accessing the dynamic transmission hook to securely forward actions and structural data packets up to store layers.
- **`useSelector()`**: Reading safe read-only slices out of global storage vectors directly into reactive interfaces seamlessly.
